# Launch checklist template

Per app launch. Copy and customize.

## Pre-launch

- [ ] App registry entry in `apps/{id}.json`
- [ ] Store listings (iOS + Android)
- [ ] Privacy policy URL live
- [ ] Support email configured
- [ ] Screenshots and captions
- [ ] Landing page with UTM-tracked store links
- [ ] App Store Connect / Play Console analytics enabled

## Branding (AI-assisted)

- [ ] Brand name / username suggestions reviewed
- [ ] Profile bios per platform
- [ ] Banner and logo concepts
- [ ] Link page (Linktree-style or custom)

## Social accounts (manual)

- [ ] Decide: developer brand vs dedicated app account
- [ ] Create accounts on chosen platforms (manual verification)
- [ ] Connect official APIs where available

## Content (AI-assisted, human-approved)

- [ ] First 30 posts drafted
- [ ] Content calendar for launch week
- [ ] Reddit / community post suggestions
- [ ] Press release draft (optional)

## Post-launch

- [ ] Track sources weekly (store consoles + UTM)
- [ ] Note which channels convert vs bounce
- [ ] Iterate listing copy based on learnings
